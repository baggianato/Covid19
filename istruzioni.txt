ISTRUZIONI DI BASE GIT
https://www.youtube.com/watch?v=SWYqp7iY_Tc&t=951s

git init - inizializza la repository
git status - stato della repository
git add <file> - aggiunge un file per un commit
  pars: . (all); *.est_file (tutti i file .est_file)
git commit -m 'commento sul commit'

prima push:
  git remote add origin "url della repo su github" - pronti a pushare
  git push -u origin master - pusha

poi:
  git push - pusha
